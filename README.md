# Get information about IP in C 
tested on Arch linux + GCC 10.2.0 + GNU Make 4.3

Dependencies:
```
make
gcc
```

Build and install:
``` bash
$ make
$ sudo make install
```

Clean:
``` bash
$ make clean
```

Uninstall:
``` bash
$ sudo make uninstall
```

Usage:
``` bash
$ ip-info {IP or host}
$ ip-info # for get your ip detail (No argument).
```
